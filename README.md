# gameshopDB
6th DB developed for the Advanced Databases Course

## This database is designed for the management and handling of inventories, sales, branches, and customers for a video game store. It is built directly using SQL as requested for the assignment.

# DATABASE FEATURES
1. Inventory Management: A database system to maintain stock control.
2. Platform Support: Management of games based on their respective gaming platforms.
3. Sales System: Detailed transaction records including various payment methods.

# DATABASE STRUCTURE
1. VIDEO GAMES: Includes title, genre, and price.
2. STOCK: Establishes the relationship between video games and physical inventory across different branches for precise control.
3. CUSTOMERS: Records customer data linked to their sales receipts.
